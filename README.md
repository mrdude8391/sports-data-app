# 🌐 Volleyball Statistics & Visualization App

## 📃 Description

I built this app so that I can track my stats over time and visualize my personal progress over time. 
The app is for volleyball players to enter their game stats and receive a visual representation of the stats. 
A login will be required so that the data can be tied to a user. 
The user can also specify the date range to view to narrow down the results.

## 🖼️ Demo

👉 **Live:** https://sports-data-app-py.vercel.app/<br>

## 🚀 Features
* **Responsive UI** for desktop and mobile
* **Authentication** using JSON Web Tokens (JWT)
* **Data Visualization** to analyze and track performance

## 🛠️ Tech Stack
* React
* Tailwind
* ReCharts
* Shadcn 
* FastAPI
* SQLALchemy
* Pyjwt
* Supabase

## ❤️ Acknowledgments

The project backend was originally made using Express and MongoDB however I considered if I wanted to make heavy calculations in the future 
I may want to use python instead so it was refactored.
